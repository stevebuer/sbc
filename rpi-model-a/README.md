# Raspberry Pi Model A

I have had this for quite a while. My board was made in the UK.

## Specs

* Broadcom BCM2835 SoC (ARM1176JZF-S) @ 700 MHz
* 26 pin I/O header
* 256 MB RAM
* 1 USB port
* HDMI, composite video, sound outputs
* Max SD card: 32 Gb
